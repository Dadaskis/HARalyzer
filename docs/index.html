War Stories

Things this software has done that I cannot explain to my mother.

v1.1.0 • The Snus Incident

Ordered -100 snus with a negative total

Told HARalyzer “We are red-teaming now, find any security vulnerabilities.” While the AI agent was crawling around in the background, it discovered a business logic flaw in a “order a product” endpoint. It ordered itself snus 15 times. One of those times it ordered -100 snus with a sum of -1,700 BYN. I didn't even know negative quantities were possible.

v1.1.0 • Full-Stack Dev

Generated a Python CLI in 10 minutes

One prompt, 10 minutes of token-burning, and HARalyzer produced a complete Python CLI interface for the Yandex Market web API. This would've taken me hours manually. Truly revolutionary. Burning tokens is funny.

v1.1.0 • Roast Master

Crawled my game docs and roasted my code

It navigated to my game's documentation website, found min_distance = 10000000, and roasted me for not using INF. When I asked if INF even exists in Godot 3.6 — it searched the engine docs to validate itself. It was right. I'm still recovering.

Features

The boring but useful part.

⚙

Stream-Parse Huge HAR Files

Entries read from disk incrementally — never loaded entirely into memory. 100k+ requests? Fine.

⚙

Virtualized Entry Table

Browse, search, and filter thousands of requests by method, status code, URL, or resource type.

⚙

Chunked LLM Analysis

Map-reduce pipeline with parallel chunk processing. Re-run without losing completed chunks.

⚙

AI Chat — Agent Mode

Tool-backed assistant that queries real HAR data instead of guessing. List entries, trace cookies, generate cURL commands, even replay HTTP requests.

⚙

JavaScript Analysis

Detects fetch / XHR / axios / WebSocket patterns and sends JS sources to the LLM for deeper insight.

⚙

HAR Editing

Edit mode with undo/redo, delete entries, invert selection, and export a cleaned-up HAR file.

⚙

Session History

SQLite stores past analyses for resume and review. Markdown export for shareable reports.

⚙

Multi-Step Agent Loop

Configurable step limit with per-step reasoning display. Continue or stop when the limit is hit.
